# 🛒 Retail Sales Analysis using MySQL

A SQL portfolio project that analyzes retail sales data to uncover business insights using MySQL. This project demonstrates data exploration, cleaning, aggregation, filtering, and subqueries to answer real-world business questions.

---

## 📖 Project Overview

This project analyzes a retail sales dataset containing **1,000 transactions** across three product categories. The goal is to explore the dataset, validate data quality, and answer key business questions using SQL.

---

## 📊 Dataset Information

| Feature | Details |
|---------|---------|
| Total Records | 1,000 |
| Columns | 9 |
| Categories | Electronics, Clothing, Beauty |
| Database | MySQL |

### Dataset Columns

- Transaction ID
- Sale Date
- Customer ID
- Gender

- Category
- Quantity

- Total Amount

## ❓ Business Questions Solved

- What is the total revenue?
- What is the average order value?
- Which product category generated the highest revenue?
- How many transactions were made by each gender?
- What are the top 5 highest-value transactions?
- Who are the highest-spending customers?

---

## 📈 Key Insights

- 💰 **Total Revenue:** ₹456,000
- 🛒 **Average Order Value:** ₹456
- 🏆 Electronics generated the highest revenue.
- 👩 Female customers accounted for **51%** of all transactions.
- 📊 The highest transaction value recorded was **₹2,000**.
